### Installing Software

Software installation uses a **package manager**. Downloading/installing software should use package manger, which is like an App Store for your phone. `apt` is a widely used package manager for Debian-based systems, such as Ubuntu.

**Example**: Install `htop` (system monitor). It's like a dashboard that gives you a real-time view of what's happening inside your computer.

```bash
sudo apt update
sudo apt install htop
```
**In screenshot, htopÂ shows:**
* **Top**: CPU and memory usage, as well as how long your computer has been running (uptime).
* **Middle**: A list of all the running programs (processes).
* **Bottom**: Options for interacting withÂ htop.

ğŸ‘‰ Use arrow keys to navigate, `q` to quit htopÂ when you're done.

## Basic File Operations

* `cd ..` â†’ Move up one level
* `ls ..` â†’ List contents of parent directory
* `ls ~` â†’ List home directory
* `touch file1.txt` â†’ Create empty file (or update timestamp if it exists)
* `echo "Hello, Linux" > file2.txt` â†’ Write to file
* `echo "Hidden file" > .hiddenfile` â†’ Create hidden file
```bash
    ls > file.txt # Overwrites file.txt with the output of 'ls'
    echo "New line" >> file.txt # Appends "New line" to the end of file.txt
```
### Listing Files

* `ls -l` â†’ Long format (permissions, owner, size, date)
* `ls -a` â†’ Show hidden files

### Move / Rename

* `mv file1.txt newfile.txt` â†’ Rename
* `mv newname.txt testdir/` â†’ Move file to directory
* `mv testdir_copy new_testdir` â†’ Rename directory
* `mv testdir/newname.txt ./original_file1.txt` â†’ Move + rename

### Remove

* `rmdir dirname` â†’ Remove empty directory
* `rm file` â†’ Remove file
* `rm -rf dir` â†’ Recursive & force delete (**âš  dangerous**)

âš âš âš  **Be careful with `rm -rf`.** Example:

```bash
rm -rf /
```

A small typo could delete critical system files or your personal data. There is no undo. For example,Â rm -rf /Â could attempt to delete your entire system (if you have permissions).Â Always double-check the path.

**Other Useful commands**

* `pwd` â†’ Print working directory
* `echo ~` â†’ Display home directory
* `cp -r` â†’ Copy files/directories recursively

* `head -c1 file.txt` â†’ First byte

* `tail -c2 file.txt` â†’ Last 2 characters

### Compare Files

```bash
diff file1 file2
```

```
1c1
< this is file1
---
> this is file2
```
Lines starting with <: Indicate lines present only in the first file (file1).
Lines starting with >: Indicate lines present only in the second file (file2).
Lines starting with ---: Act as a separator between changes in the first and second files when a line has been modified.
